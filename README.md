# Portofolio Network Engineer

Website portofolio profesional untuk seorang Network Engineer dengan tema hitam dan aksen hijau neon.

## Fitur Utama

1. **Desain Responsif** - Tampilan optimal di desktop, tablet, dan mobile
2. **Hero Section dengan Foto** - Layout dengan foto profil di sebelah kanan dan informasi di kiri
3. **Warna Tema** - Dominan hitam dengan aksen hijau neon (#00ff9d) untuk tema network engineer
4. **Bagian-bagian Penting**:
   - Header dengan navigasi responsif
   - Hero section dengan statistik dan tombol CTA
   - Tentang saya dengan skill tags
   - Galeri proyek jaringan
   - Sertifikat profesional
   - Form kontak yang fungsional
   - Footer dengan tautan media sosial

5. **Efek Visual**:
   - Animasi jaringan di background (canvas-based)
   - Efek hover pada kartu proyek dan sertifikat
   - Transisi halus saat scroll
   - Efek typing pada judul hero

6. **Interaktivitas**:
   - Menu mobile toggle
   - Form kontak dengan validasi
   - Smooth scrolling untuk navigasi
   - Notifikasi saat form dikirim

# Setup Form Kontak dengan EmailJS

## Langkah-langkah Setup:

### 1. Daftar EmailJS Gratis
1. Buka [https://www.emailjs.com](https://www.emailjs.com)
2. Klik "Sign Up Free"
3. Daftar dengan email Anda (mendukung 200 email/bulan gratis)

### 2. Setup Service Email
1. Setelah login, klik "Email Services" di sidebar
2. Klik "Add New Service"
3. Pilih penyedia email Anda (Gmail, Outlook, dll)
4. Hubungkan dengan email Anda yang ingin menerima pesan
5. Catat **Service ID** yang diberikan

### 3. Buat Email Template
1. Klik "Email Templates" di sidebar
2. Klik "Create New Template"
3. Buat template dengan konten seperti:
